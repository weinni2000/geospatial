Installation
============

Take a look at the installation section in the description of the module
'base_geoengine'.

Usage
=====

-  Open a partner form
-  Go in Geoengine Map tab
-  Change to edit mode
-  Click on the map to add a point.

To remove the point click on the trash bin. To edit the point drag and
drop it.
